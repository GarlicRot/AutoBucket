## Features

- Automatically captures nearby aquatic mobs with a water bucket
- Configurable settings for targeting specific mob types:
  - Axolotls
  - Cod
  - Pufferfish
  - Salmon
  - Tadpoles
  - Tropical Fish
- Cooldown system to prevent rapid capturing
